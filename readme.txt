Role wise category post lets you see posts based on the category privillages for users. 

== Description ==


This plugin gives access to users based on his/her selected category post. It also provides shortcode for the list of selected category.

Features:
-	Easy installation
-	Admin have rights to set privileges for a particular category post that has to be seen by the user on frontend.
-	Shows only selected category lists in front-end. (For that use short code). 
    [selected_category_list]


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Install the plugin via WordPress or download and upload the plugin to the /wp-content/plugins/
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can see the 'Restrict category post By Admin' menu.
